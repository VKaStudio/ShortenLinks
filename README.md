# ShortenLinks
Small service for shortening links | PHP + JS + SQL

<h2>Dependencies:</h2>
<li>jquery</li>
<li>webserver (php)</li>
<li>Database (MySQL)</li>

<h2>Opportunities:</h2>
<li>Generation of short links.</li>
<li>When clicking on a shortened link, redirection to full links.</li>
<li>Displays all generated links.</li>
<li>Keeping statistics of redirects.</li>
